## 四、测试执行指南

### 4.1 后端测试执行

```bash
# 进入后端目录
cd backend

# 创建虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

# 运行所有测试
pytest tests/ -v

# 运行指定测试文件
pytest tests/test_auth.py -v

# 运行并生成覆盖率报告
pytest tests/ -v --cov=app --cov-report=html
```

### 4.2 移动端测试执行

```bash
# 进入移动端目录
cd mobile

# 获取依赖
flutter pub get

# 运行所有测试
flutter test

# 运行指定测试文件
flutter test test/login_screen_test.dart

# 生成覆盖率报告
flutter test --coverage
```

## 六、已知限制

1. **外部服务依赖**: ASR/LLM 服务需要真实环境运行
2. **数据库**: 测试使用 SQLite 内存数据库，与生产 PostgreSQL 可能存在差异
3. **音频测试**: 录音功能需要在真实设备上测试
4. **权限测试**: 麦克风权限需要在真机上测试
5. **网络测试**: 离线场景需要在设备上模拟
